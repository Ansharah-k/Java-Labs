# ☕ Java Programming Labs

A collection of Java lab exercises completed as part of my programming coursework.

---

## 🧪 Lab Summaries

### 🔹 Lab 1 — Introduction to Java
- Basic Java program structure
- Printing output using `System.out.println()`

### 🔹 Lab 2 — Data Types & Variables
- Working with `int`, `double`, `float`, `char`, `boolean`, `String`
- Variable declaration and initialization

### 🔹 Lab 3 — Operators, Type Casting & User Input
- Taking user input using `Scanner`
- Arithmetic operations and type casting
- Escape sequences (`\n`, `\"`, `\t`)
- Built billing and salary management systems

### 🔹 Lab 4 — Conditional Statements
- `if`, `else if`, `else` statements
- Nested conditions

### 🔹 Lab 5 — Loops
- `for`, `while`, `do-while` loops
- Loop control statements

### 🔹 Lab 6 — Arrays
- Single and multi-dimensional arrays
- Array traversal and manipulation

### 🔹 Lab 7 — Methods / Functions
- Defining and calling methods
- Parameters and return types

### 🔹 Lab 8 — Object Oriented Programming
- Classes and Objects
- Constructors and attributes

### 🔹 Lab 9 — Math Quiz Game
- Switch-case statements
- Random number generation using `Math.random()`
- While loop for play again feature
- Difficulty levels: Easy (+), Medium (*), Hard (/)

---

## 🛠️ Technologies Used
- **Language:** Java
- **IDE:** NetBeans / VS Code
- **JDK Version:** Java 8+
